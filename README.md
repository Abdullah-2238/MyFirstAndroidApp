# MyFirst Android App 

A native Android application built with Java as part of my Android
development learning journey.

##  Features
- User input handling with EditText
- Dynamic UI updates with TextView
- Button click interactions
- Clean LinearLayout UI design

##  Built With
- Java
- Android SDK
- Android Studio

##  Concepts Practiced
- Activity Lifecycle
- findViewById() view binding
- OnClickListener interface implementation
- XML Layout Design
- LinearLayout, TextView, EditText, Button

##  Getting Started
1. Clone the repo
2. Open in Android Studio
3. Run on device or emulator
